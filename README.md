# 🎬 Netflix User Behavior Analysis

An end-to-end Data Analytics project that analyzes Netflix user behavior using **Databricks, PySpark, SQL, Machine Learning, and Power BI** to generate actionable business insights.

---

# 📌 Project Overview

This project analyzes Netflix user engagement, subscription patterns, viewing habits, customer churn, and user segmentation. Data was processed in Databricks using PySpark, analyzed with SQL, clustered using Machine Learning (K-Means), and visualized through interactive Power BI dashboards.

---

# 🎯 Business Objectives

- Analyze user engagement and viewing behavior.
- Identify high-value and low-engagement users.
- Understand customer churn trends.
- Segment customers using Machine Learning.
- Build interactive dashboards for business decision-making.
