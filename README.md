# Netflix User Behavior Analysis

## 📌 Project Overview

This project focuses on analyzing Netflix user behavior, understanding engagement patterns, identifying customer segments, and gaining insights into churn behavior.

An end-to-end data analytics solution was developed using Databricks, PySpark, SQL, Machine Learning, and Power BI to transform raw user data into meaningful business insights.

---

## 🎯 Business Objective

- Analyze user engagement and viewing behavior
- Identify different user segments based on activity patterns
- Understand customer churn trends
- Create an interactive dashboard for business decision-making

---

## 🛠️ Tools & Technologies

- Python
- PySpark
- Databricks
- SQL
- Machine Learning
- Power BI
- Pandas
- Matplotlib
